
# Rekuta

A simple UTAU voicebank recorder app for android. "Rekuta" stands for **Rec**ord **UTA**U.

## Main Features

- Lossless audio recording
- Simple UI
- CV and VCV support (more soon)

## Libraries

- Google GSON
- OmRecorder
